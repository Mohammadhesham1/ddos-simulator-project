# DDoS Simulator App (Cyber Security Graduation Project)

هذا التطبيق هو مشروع تخرج مخصص لمحاكاة هجمات **DDoS (Distributed Denial of Service)** لأغراض تعليمية واختبار أمن المواقع.

## الهدف من المشروع
- اختبار قدرة المواقع على تحمل عدد كبير من الطلبات المتتابعة.
- مساعدة فريق التأمين (Blue Team) على تجربة أدوات الحماية وتكوين جدران الحماية (Firewalls).

## المميزات
- واجهة بسيطة لإدخال رابط الموقع المستهدف.
- استخدام **Kotlin Coroutines** لإرسال طلبات HTTP متتابعة بسرعة عالية.
- عرض حي لعدد الطلبات المرسلة وحالة الاستجابة من السيرفر.

## كيفية الحصول على APK
يتم بناء ملف الـ APK تلقائياً عند كل تحديث للكود عبر **GitHub Actions**. يمكنك تحميله من تبويب **Actions** في هذا المستودع.

## تنبيه هام
هذا التطبيق مخصص للأغراض التعليمية فقط ضمن المبادرة الحكومية. استخدام هذا التطبيق ضد مواقع بدون إذن مسبق يعتبر عملاً غير قانوني.
